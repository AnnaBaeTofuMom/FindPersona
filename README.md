## 코드 컨벤션

**커밋**

- [`커밋종류`] 파일명 : 변경내용
    - **ADD** : 새로운 내용 추가
    - **MOD** : 기존 내용 변경
    - **DEL** : 기존 내용 삭제
- ex
    
    ```markdown
    [ADD] MainView : 신규 뷰 생성
    ```
    

**브랜치**

- 브랜치 종류
    - main
    - develop
    - feature
    - release

- feature/`이슈타이틀`
- ex
    
    ```markdown
    feature/컨벤션정립
    ```
    

**PR**

- [PR][PR에 대한 요약]
- ex
    
    ```markdown
    [PR] 코드 컨벤션 정리 완료
    ```
